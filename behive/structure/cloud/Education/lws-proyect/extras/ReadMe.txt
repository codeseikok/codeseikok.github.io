This directory contains some extra classes that can be used with SPL.

AIRMenuHelper
Inserts standard edit options in AIR context menus (they are lost by default when setting custom context menus).

RegExpHighlighter & RegExpHighlight
Provides capabilities to highlight regular expressions in a text field. Good example of extending the TextHighlighter class.
Used in RegExr (regexr.com).

gWordListEditor - Allows you to edit, or create new word lists for use with SPL.
